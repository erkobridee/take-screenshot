ScreenShot service powered by Nodejs, Hapi.js and PhantomJS


/api/screenshots [POST] 
request:
{
  url: String,
  screenshot: String,
  resize: String,
  delay: Number
}

> url        : screenshot target url
> screenshot : screenshot width x height x top ( default: 1024x768x0 )
> resize     : resize screenshot image to width x height ( default: 250x200 )
> delay      : time in miliseconds to take screenshot ( default: 300 )


response:
{
  id: String
}

> id : screenshot image id

/api/screenshots/{id} [GET] 
> id : screenshot image id

return screenshot image file
